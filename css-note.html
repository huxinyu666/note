CSS基本概念

  1. CSS: 级联样式表 Cascade Style Sheet
  2. css 基本由多个rule组成
  3. 每一个rule由 selector(选择器)和属性及属性值组成
  4. Selector: 表达方式：
    语法 举例
    #id 作用于含有该id的元素
    .classname .big
    tag div
    selectorselector .big.middle 两个selector连着写，作用于同时拥有这两个class的元素
    selector selector .big  .middle 两个selector隔着一个空格，则该样式作用于.big下面所有.middle
    > .big > .middle 两个selector隔着一个>, 则该样式仅作用于父亲是.big的.middle
    [border] 作用于所有有border属性的元素
