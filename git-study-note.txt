git study note 

Git是什么
Git是保存程序源代码，并提供一系列代码维护功能的网上仓库。

Github是什么
github.com 是一个网站,它提供git服务 可以把源代码存到这个网站上面,也可以学习其他人的代码

URL是什么？
Uniform Resource Locator
是一个标准 有一定格式 为了标记某一个资源的位置


怎么把代码传到Git上？
1. 在Git上建立一个Repository(项目)
2. 在本地把这个Repository拉下来
   git clone + 项目的Git URl， eg. https://github.com/huxinyu666/note.git
3. 添加新文件或做出一些改动。
3.5 用git status(gst)查看当前状态
4. 用git add . 确认你的改动
5. 用git commit -m 提交并写出改动的标题
6. 用git push origin(gp) 上传改动
